Xunzhu Tang



Daniel (Xunzhu) is a doctoral researcher of Trux, Interdisciplinary Security and Trust Centre (SnT), University of Luxembourg (UL). His research interests mainly relate to Code&Patch Explanation, Automatic Program Repair, Collaborative Agents for SE, Conversational IDE, Security in Large Language Model (LLM) (i.e, LLM poisoning, LLM watermarks). His researches has been published in largely diverse top-tier conferences/journals in both NLP, SE, and ML communities, including ICSE, ASE, EMSE, ICLR, ACL, NAACL, etc. Furthermore, Daniel also work as PC/Reviewer/AC in SOSP, ACL, EMNLP, MSR, etc.

Research interests:

  • LLM-Based Agents for software engineering: CodeAgent, SynFix, Reparity, etc.
  • Patch Representation and Explanation: Patcherizer, LLMDA, etc
  • Software Security
  • Automatic Programm Repair
  • Large Code(Language) Model
